Mars Rover

The Rover is facing North at position [4,4] by default, in a 10 x 10 map grid.

In the console, input a command string using the Command("your_command_string"); function, with an argument of a string composed by: (f)orward, (b)ackward, (l)eft or (r)ight.

Example:

Command("rffrbfflfrbff");

Note: any other letters will not be recognized by the Rover
